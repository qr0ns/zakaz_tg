# 📊 Telegram Quiz Bot — бот для тестирования

Telegram-бот для проведения тестов и автоматического подсчёта результатов.

## 🚀 Возможности

* Прохождение тестов в Telegram
* Использование FSM (сценарии)
* Inline-кнопки для ответов
* Подсчёт результатов в реальном времени
* Выдача итогового результата пользователю

## 🛠 Технологии

* aiogram (FSM, StateGroup)
* Telegram Bot API

## 📸 Пример работы

Пользователь проходит тест → выбирает ответы → бот считает результат → выдаёт итог (например: прошёл / не прошёл)

## ⚙️ Установка

```bash
git clone https://github.com/qr0ns/telegram-quiz-bot
cd telegram-quiz-bot
pip install -r requirements.txt
cd app
python main.py
```

## 🔑 Переменные окружения

```
BOT_TOKEN=your_token
```

## 💼 Применение

* Онлайн-тесты
* Образовательные платформы
* HR-отбор (тестовые задания)
* Квизы и опросы
